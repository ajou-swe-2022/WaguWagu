## Atomic Pattern

- Atom : 가장 작은 Component 단위
- Molecules : 하나의 통합된 작업을 수행하는 Component 묶음
- Organism : 배치를 위한 하나의 묶음
- Template : 배치를 위한 공간 구분
- Page : 하나의 페이지
